# 🛍️ Nostra - E-Commerce Website

Nostra is a responsive e-commerce website built using **HTML, CSS, Tailwind CSS, and JavaScript**.
It showcases a modern UI with product listings, search functionality, and multiple pages.

---

## 🚀 Features

* 🔍 **Search Functionality**
  Filter products dynamically using the search bar.

* 🧥 **Product Collection Page**
  Displays multiple products in a grid layout.

* 📱 **Responsive Design**
  Works on mobile, tablet, and desktop devices.

* 📄 **Multiple Pages**

  * Home Page
  * Collection Page
  * Contact Page

* 🎨 **Modern UI**

  * Clean layout
  * Tailwind CSS styling
  * Hover effects and transitions

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **Tailwind CSS**
